# Image Asset Guide

1. 이미지 파일명은 영문 소문자와 언더스코어만 사용합니다.
2. 한글 파일명은 사용하지 않습니다.
3. 홈 배너는 1600x900 WebP를 권장합니다.
4. 문제풀이 이미지는 1200x800 WebP를 권장합니다.
5. 완료/결과 이미지는 1400x1400 WebP를 권장합니다.
6. 캐릭터 단독 이미지는 1024x1024 PNG 투명 배경을 권장합니다.
7. 배지/스티커는 512x512 PNG 투명 배경을 권장합니다.
8. 이미지 안에 텍스트를 넣지 않고 앱 코드에서 Text로 표시합니다.
9. 제철배움앱 이미지는 같은 캐릭터, 같은 감성, 같은 세계관을 유지합니다.
